TYPES OF WOOD TREES.

a) Hardwood trees.

These are trees whose wood lasts for a long time.

Examples of hardwood trees.

  • Mvule
  • Mahogany
  • Eucalyptus
  • Mugavu
  • Teak
  • Misambya
  • Oak

b) Soft wood trees.

These are trees whose wood do not last for a long time.

Examples of soft wood trees.

  • Kirundu
  • Enzingu
  • Wattle
  • Misizi
  • Mutuba (ficus)
  • Jackfruit tree

NOTE: The bark of Mutuba tree is used to make bark clothes

Ever green trees. 

These are trees which bear cones (coniferous) e.g. Fir, pine, cedar.
