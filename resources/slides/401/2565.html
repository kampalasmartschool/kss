Common abbreviations

Latin abbreviations

  1. A.D – Anno Dmini ( In the year of the lord
  2. A.m – ante meridian. ( before noon)
  3. P.M – Post meridiem. ( afternoon)
  4. E.g. – example gratia ( for example )
  5. Etc – et cetera ( and so on)
  6. i.e – id est ( that is )
  7. N.B – Note bene ( not well)
  8. R.S.V.P –Respondez sil vous plait ( please reply)
  9. Viz – Videlicet ( namely)
  10. Via – by way of
