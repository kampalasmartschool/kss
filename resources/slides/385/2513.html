Materials which affect light are grouped into:

  1. Transparent objects.
  2. Translucent objects
  3. Opaque objects.

i. Transparent objects

These are objects which allow most of the light to pass through and we can see through them.

Examples of transparent objects.

  • Glass
  • Water
  • Air etc.

ii. Translucent objects

These are objects which allow some light to pass through but we cannot see through clearly.

We cannot see through them because they diffuse or scatter light rays in all directions.

Examples of translucent objects.

  • Frosted glass.
  • Waxed paper
  • Cloth
  • Tissue paper
  • Light bulbs.

iii. Opaque objects.

These are objects which don’t allow any light to pass through them.

  • We cannot see through them because light travels in straight lines.
  • Opaque objects instead form shadows.

Examples of opaque objects.

  • Wood
  • Stones
  • Metals.
  • Walls.
  • Bricks etc.
