APOSTROPHE ( ’  )

  • Used to replace an omitted letter e.g.  can’t, don’t  etc .
  • used to indicate ownership or possession  e.g. Mary’s dress,  Uganda’s  etc
  • Use to form the plural of letters, signs and figures e.g.  Mary writes her  ts’ correctly
  • For nouns which end with letter ‘s’ we only add the apostrophe e.g. Moses’ shirt,  Jesus’ name

Examples

1. They’ve arrived.

2. Joel’s dad was around today.

3. Jade spoilt James’ toys.
