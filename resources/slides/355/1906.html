TYPES OF SOIL.

There are three types of soil namely:

  1. Loam soil.
  2. Clay soil.
  3. Sandy soil.

LOAM SOIL.

It has the following properties.

  1. It contains balanced particles of sandy and clay.
  2. It contains more humus than clay and sandy.
  3. It is well aerated than clay.
  4. It contains all soil components in a balanced amount.

NOTE: Therefore it is the best for crop growing.

CLAY SOIL.

It has the following properties.

  1. It has little humus
  2. It has fine particles
  3. It is poorly aerated
  4. It retains a lot of water
  5. It is sticky and heavy
  6. It is suitable for pottery.

SANDY SOIL.

It has the following properties.

  1. It is well drained
  2. It has poor rate of capillarity
  3. It has large and rough particles
  4. It is well aerated
  5. It dries quickly
  6. It is suitable for construction
