Elements of a good map.

  • Map key
  • Map title
  • Map compass direction / a compass rose
  • Map scale
  • Map frame.

1. Map Key.

It is a table bearing symbols used on a map and their meaning.

The main function of a key is to interpret symbols and signs used on a map.

NOTE: Symbols are signs that represent real objects on a map.

Importance of Map Symbols.

  • They prevent over crowdedness on a map
  • They make the map tidy
  • They make map reading easy.

Examples Of Map Symbols.

2. Map Title.

It is the name or heading of a map.

It helps us to know what the map is all about.

3. Map Compass direction.

It is a symbol that is used to show direction of places on a map.

Primary/cardinal points.

  • North
  • East
  • West
  • South

Secondary points.

  • North West
  • North East
  • South West
  • South East

People (Groups of people) who use a compass:-

  • Sailors.

  • Guides.

  • Pilots.

  • Mountain climbers.

  • Tourists.

  • Astrounants.

  • Soldiers / army.

  • Surveyors.

4. Map Scale.

Scale helps to determine the actual distance between places on a map.

Type of scales

a) Linear Scale.

b) Representative/fractional Scale.

E.g: 1:1000km

c) Statement Scale.

1cm represents 50km

OR:  

1cm = 50km

Using a scale to find Distances.

Example:

Yole moved from town A to B a distance of 5cm.  Given that 1cm represents 50km (1:50km) Find the actual distance.

Soln:

A<----5cm------>B

1cm   =       50km

5cm   =       (5 x 50)km

          =       250km.
