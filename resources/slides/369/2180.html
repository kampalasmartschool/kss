If 3

If 3 is used to express what would have happened if a certain condition had been fulfilled. We use the past perfect tense in the if clause and would/could/might/should + have and the past participle verb in the main clause.

Example

If I had had money, I would have gone to London.

If he had seen me yesterday, he would have told me.

If he had not warned me, I would have made a mistake.

NB:“had” is used to emphasize the supposition.

Had can be used to start an if 3 when if is left out or omitted.

Examples

  • Had I seen him during the holidays, he would have given me some money.
  • Had we gone to the village, we would have seen our grand parents.

When the main clause is negative, we use the grammatical order below.

Would not have

Could not have

Should not have

Might not have

Example

If Mary had helped her parents, they would not have punished her.
