PUTTING SETS ON A VENN DIAGRAM.

Examples:   

X  =  {1, 2, 3, 4, 5}

Y  =  {0, 2, 4, 6, 8}

Represent the two sets on a Venn diagram.

List members of:-

i. X only = {1, 3, 5}

ii. Y – X  = {0, 6, 8}

iii. X ∩Y =  {2, 4}

SUBSETS.

Definition.

A subset is a set of members got from a given set.

An empty set is a subset of any set.

A set is a subset of itself (it’s called a super set). 

Listing subsets.

Set P = {1, 2, 3}     

The subsets are:-

{ }, {1, 2, 3}, {1, 3}, {2, 3},  {1 }, {2 }, {3 }, {1, 2 }
