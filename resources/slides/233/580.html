Finding the number of elements in intersection sets.

 

Example 1.

M = { a , b , c , d , e } and   N = { a , e , i , o ,u }.

Find n (M  N)

 M = { a , b , c , d , e } and   N = { a , e , i , o ,u }.

                     M N = {a , e}

 

                         n (M  N)=2   

 Example 2.

Set A = { d , i , g }and set   B = { d , o n , e }. How many members are in set A  B?

 A = { d , i , g }        B = { d , o n , e }.

 

                      A B  = { d }

      There is one member in set A  B.

    OR n(A  B)= 1.
