Finding the number of members in the intersection set using symbol (n)

e.g.

1)

P = {a, b, c}                                                                  

 Q = {c, f, a}                                                                 

Find n (P∩Q)                                                               

P∩Q = {a}                                                          

n(P∩Q) = 1 member                                             

2)

A = {1, 2, 3, 5}

B = {2, 3, 5, 7, 9}

Find n(A∩B)

A∩B = {2, 3, 5}

n(A∩B) = 3 members
