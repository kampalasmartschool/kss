INTERSECTING SETS “”

 

Intersecting sets are sets with common members or elements.

          

           Example:-

1. Set  F = {e , a , r} and set   G = { l , e  , a , n }

         F = {e , a , r}            G = { l , e  , a , n }

           Common members ={e , a }

 F G= {e, a}

  • Common members are written only once.

Example 2.

           Set A = { d , i , g }and set   B = { d , o n , e }

A = { d , i , g }and set   B = { d , o n , e }

            A  B  = { d  }
