Sets.

What is a set?

  • A set is a collection of well-defined objects.
  • Things found in a set are called members or elements.

Example of sets.

a) A set of letters.

        

b) A set of 3 trees.


TYPES OF SETS.

1). Equal sets     

2). Non-equal sets         

3). Equivalent sets 

4) Non – equivalent sets

5). Empty sets   

6). Intersecting sets       

7). Union set.
