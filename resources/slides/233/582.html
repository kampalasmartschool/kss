Using Venn diagrams to solve set problems.

Example 1 :-

Study the Venn diagram below and answer the questions that follow;

a) List down all the elements of set A.

Set A = {a, b, c, d}

b) List the members of set B.

Set B = {d, e, f} 

 

c) Find;

i) A  B. 

A B = {d} 

 

ii) A U B. 

A U B = {a, b, c, d, e, f}

 

Example 2.

Study the Venn diagram below and use it to answer the questions that follow;                              

                          

 

 

a) Find n(J)

Set J={0, 1,2, 3, 4}

n(J)=5

 

b) How many members are in set K?

Set K={0,4,5,6}

There are 4 elements in set K.

OR;   n(K) =4

 

c) n(J  K).               

J  K ={0,4}                   

n (J  K) =2

 

d)  List down all the elements in set J U K.

J U K={0,1,2,3,4,5,6}.
