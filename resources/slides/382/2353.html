Opposites from prefixes (un, dis, mis, in, irr, il etc)

A prefix is a letter or group of letters added to the beginning of a word on order to change its meaning.

Using prefix “ir”

Regular                           irregular

Responsible                    irresponsible

Relevant                          irrelevant

Using prefix “im”

Polite                                impolite

Possible                           impossible

Passable                          impassable

Pure                                 impure

Patient                             impatient

Movable                           immovable

Perfect                             imperfect

Using prefix “un”

Happy                              unhappy

Tidy                                  untidy

Reliable                           unreliable

Married                            unmarried

Common                          uncommon

Skilled                              unskilled

Healthy                             unhealthy

Popular                             unpopular

Kind                                  unkind

Using prefix “il”

Legal                                  illegal

Literate                               illiterate

Legible                                illegible

Logical                                illogical

Using the prefix “dis”

Like                                     dislike

Obedient                            disobedient

Honest                                dishonest

Appear                                disappear

Satisfied                              dissatisfied

Arm                                     disarm

Agree                                  disagree
